# Abstract Draft

Clinical narratives describe evolving beliefs about diagnoses, symptoms, tests, and treatments. Existing clinical NLP work often focuses on entity extraction, assertion classification, or temporal event extraction, but these formulations do not directly model how new textual evidence updates the epistemic state of a clinical proposition. We introduce MedEST-Open, an open prototype task for Medical Epistemic State Tracking, where each example represents a proposition, its previous state, a new evidence segment, a semantic transition operator, and an updated state. We propose EpiDelta inputs that combine old state, proposition text, and evidence text before embedding with PubMedBERT. On weak labels derived from the open MACCROBAT case-report dataset, a balanced logistic regression model over EpiDelta embeddings achieves 0.7369 accuracy and 0.6641 macro F1 on a 7-way filtered transition prediction task under case-level split. Error analysis highlights clinically meaningful confusions such as introduced versus treated and improving versus resolved. These results suggest that evidence-conditioned state-update modeling is a promising direction for clinical narrative understanding.
