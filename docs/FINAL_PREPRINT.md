# Final Preprint Package

The stable preprint version is stored under `preprint/`.

- PDF: `preprint/MedEST_Open_final_fixed_preprint.pdf`
- arXiv source package: `preprint/MedEST_Open_final_fixed_arxiv_source.zip`

This version uses a single-column LaTeX article format to avoid two-column figure/table collisions. The author block is manually formatted to avoid affiliation-footnote problems. The paper keeps the result claim precise: MedEST-Open is a weakly supervised benchmark prototype, not an expert-gold clinical benchmark.

Final result:

- Accuracy: 0.9061
- Macro F1: 0.9075
- Weighted F1: 0.9062
- Best epoch: 5
- Model: fine-tuned PubMedBERT
